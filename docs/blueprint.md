# **App Name**: Mindful Meetings

## Core Features:

- Spreadsheet Import: Upload existing spreadsheet data for treatment group therapy management.
- Data Parsing & Structuring: Parse data from the uploaded spreadsheet to structure records in memory (or the database, if Firestore is implemented).
- Session Summarization: Use generative AI to summarize key discussion points and action items from each session based on imported notes. This is incorporated as a tool that reasons whether session details are sufficient for a valid summary.
- Progress Tracking: Visually track patient progress over time using charts and graphs generated from session data.
- Report Generation: Generate comprehensive reports summarizing patient progress, session details, and overall group trends.

## Style Guidelines:

- Primary color: Soft blue (#89B4FA), evokes calmness and trust.
- Background color: Very light blue (#F0F4FF), provides a serene and uncluttered backdrop.
- Accent color: Light purple (#BFA1F5), offers a touch of creativity and individuality.
- Body and headline font: 'Inter', sans-serif with a modern, machined, objective, neutral look.
- Use clear, simple icons for navigation and data visualization.
- Maintain a clean and organized layout with clear section headings and intuitive navigation.
- Incorporate subtle transitions and animations to enhance user experience and provide feedback.
